# Python Hello World Server

This is a simple "Hello World" server written in Python using the Flask framework.

## Running the server

1. Install the dependencies:

   ```
   pip install -r requirements.txt
   ```

2. Run the server:

   ```
   python app.py
   ```

The server will be running at http://127.0.0.1:5000/ and will automatically reload when you make changes to the code.
