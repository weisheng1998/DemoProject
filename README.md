# DemoProject

Demo Project

## Project Description

DemoProject is a simple web application built using Flask. It serves as a demonstration of basic web application structure and functionality.

## Installation

To install the necessary dependencies, run:

```bash
pip install -r requirements.txt
```

## Usage

To start the server, execute:

```bash
python server.py
```

Visit `http://localhost:5000` in your browser to see the application in action.
